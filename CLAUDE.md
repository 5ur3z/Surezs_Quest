# Surez's Quest — Minecraft 1.21.1 NeoForge Quest Mod

A quest mod with player/server quests, NPC dialogue, 5 objective types, quest chains, i18n (zh_cn/en_us).

- **Build**: `./gradlew build` — JAR at `build/libs/surezs_quest-neoforge-1.21.1-{version}.jar`
- **Run**: `./gradlew runClient` / `runServer`
- **Lang**: Java 21, NeoForge 1.21.1, Gson, Minecraft Codec API

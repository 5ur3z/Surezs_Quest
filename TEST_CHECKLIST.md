# Surez's Quest — 功能测试清单

## 一、启动与基础设施

| # | 测试项 | 预期结果 |
|---|--------|----------|
| 1 | **单人游戏启动** | 正常进入主菜单，不闪退 |
| 2 | **创建/进入世界** | 正常加载，不崩溃 |
| 3 | **后台日志** | 看到 "DataLoaders initialized" 和 "Loaded X quests/NPCs/triggers/quest lines" |
| 4 | **首次运行复制默认文件** | `config/surezs_quest/` 下自动生成 quests/npcs/triggers/quest_lines 文件夹及默认 JSON |

## 六、任务进度追踪

## 七、物品提交流程

| # | 测试项 | 预期结果 |
|---|--------|----------|
| 33 | **展开任务卡 → 点 [提交物品]** | 弹出 SubmitItemScreen，只列出匹配 objective 的背包物品 |
| 34 | **选择物品** | 点击物品行切换选中（绿色高亮）/ 取消 |
| 35 | **点 [确认提交]** | 物品扣除 → 进度更新 → 自动回到 QuestScreen（不是游戏画面） |
| 36 | **验证扣除** | 服务端二次验证 slot 内容，不受篡改客户端影响 |
| 37 | **部分提交** | 背包 5 个铁锭、任务只须 3 个 → 只扣除 3 个 |
| 38 | **达标后 [提交] 消失** | 全部 objective 达标后 [提交] 自动隐藏，只剩 [领取] |

## 十一、管理指令（需 op 权限）

| # | 测试项 | 命令 |
|---|--------|------|
| 54 | **派发任务** | `/quest give <玩家> surezs_quest:collect_iron` → GUI 出现 NPC 消息 + 任务卡 |

| 59 | **ID 自动补全** | 输入任务 ID 时 Tab 自动补全 |
| 60 | **热重载任务数据** | `/quest reload` → 从 config 重读所有 JSON，不依赖 /reload |
